# 🚀 코딩 to-do-list

코딩 학습을 돕기 위해 과제와 유용한 영상 링크를 한곳에서 관리할 수 있는 팀 프로젝트 서비스입니다.

## 1. 팀 정보
- **팀원**: 김예준, 최재우
- **서비스 이름**: 코딩 to-do-list
- **한 줄 설명**: 학생 및 개발자가 과제나 학습 영상 링크를 쉽게 저장하고 관리할 수 있도록 돕는 서비스입니다.

## 2. 주요 기능 (CRUD)
- **조회 (Read)**: 저장된 과제 리스트와 영상 링크 리스트를 화면에 표시합니다.
- **저장 (Create)**: 카테고리(일반/영상)를 선택하여 새로운 항목을 추가합니다.
- **수정 (Update)**: 작성된 내용이나 제목을 실시간으로 수정할 수 있습니다.
- **삭제 (Delete)**: 완료된 과제나 불필요한 링크를 삭제합니다.

## 3. 기술 스택 및 구현 포인트
본 프로젝트는 **대덕SW마이스터고 튜터링** 과정을 통해 배운 핵심 기술들을 사용하여 구현되었습니다.

### Backend
- **Node.js & Express**: HTTP 요청(GET, POST, PATCH, DELETE)을 처리하는 서버를 구축했습니다.
- **SQLite**: 가볍고 효율적인 파일 기반 데이터베이스를 사용하여 데이터를 영구적으로 저장합니다.
- **REST API**: 프론트엔드와 백엔드가 JSON 형식을 통해 약속된 규약대로 데이터를 주고받습니다.

### Frontend
- **Vanilla JS & Fetch API**: 라이브러리 없이 순수 자바스크립트의 `fetch` 함수를 사용하여 서버와 비동기 통신을 수행합니다.
- **Dynamic DOM**: 서버에서 받아온 데이터를 기반으로 HTML 요소를 동적으로 생성하여 화면을 업데이트합니다.
- **Responsive Design**: 다양한 화면 크기에서도 깔끔하게 보이도록 CSS Grid와 Flexbox를 활용했습니다.

## 4. 데이터베이스 설계 (DB Schema)
### `list_posts` (일반 과제)
| 컬럼명 | 타입 | 설명 |
|---|---|---|
| id | INTEGER (PK) | 고유 식별자 (자동 증가) |
| title | TEXT | 과제 제목 |
| content | TEXT | 상세 내용 |
| username | TEXT | 작성자 이름 |
| created_at | DATETIME | 생성 일시 |

### `video_posts` (영상 링크)
| 컬럼명 | 타입 | 설명 |
|---|---|---|
| id | INTEGER (PK) | 고유 식별자 (자동 증가) |
| title | TEXT | 영상 제목 |
| url | TEXT | 영상 링크 (URL) |
| username | TEXT | 작성자 이름 |
| created_at | DATETIME | 생성 일시 |

## 5. 실행 방법
1. 터미널에서 프로젝트 폴더로 이동합니다.
2. `npm install` 명령어로 필요한 패키지를 설치합니다.
3. `node server.js` 명령어로 서버를 실행합니다.
4. 브라우저에서 `http://localhost:3000`에 접속합니다.
