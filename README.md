# 코딩 To-Do List 프로젝트 발표 자료

## 1. 팀명과 서비스 이름

### 팀명

최재우, 김예준

### 서비스 이름

코딩 To-Do List

### 한 줄 소개

코딩 학습 과정에서 과제와 공부 자료(영상 링크)를 효율적으로 관리할 수 있는 To-Do List 서비스입니다.

---

## 2. 사용자는 누구인가

### 주요 사용자

* 프로그래밍을 공부하는 학생
* 개인 프로젝트를 진행하는 개발자

### 사용자 문제

* 해야 할 과제를 잊어버린다.
* 유용한 강의 링크를 저장해도 나중에 찾기 어렵다.
* 학습 자료와 할 일을 한 곳에서 관리하기 어렵다.

---

## 3. 왜 필요한 서비스인가

코딩을 공부하는 과정에서는 과제, 프로젝트 일정, 참고 영상 등 관리해야 할 정보가 많습니다.

기존 메모장이나 브라우저 북마크를 사용할 경우 자료가 분산되어 찾기 어렵다는 문제가 있습니다.

코딩 To-Do List는 학습 자료와 할 일을 한 곳에서 관리할 수 있도록 하여 학습 효율을 높여줍니다.

### 기대 효과

* 학습 자료 관리 편의성 향상
* 과제 누락 방지
* 효율적인 학습 계획 수립

---

## 4. 핵심 기능 데모

### 1) 게시글 작성

* 새로운 할 일 등록
* 제목과 내용을 입력하여 저장

### 2) 영상 링크 저장

* 유용한 강의 링크 기록
* 나중에 쉽게 찾아볼 수 있음

### 3) 게시글 수정

* 과제 내용 변경 시 수정 가능
* 최신 상태 유지

### 4) 게시글 삭제

* 완료된 과제 제거
* 불필요한 자료 정리

### 데모 시연 순서

1. 게시글 생성
2. 영상 링크 저장
3. 게시글 수정
4. 게시글 삭제
5. 목록 자동 갱신 확인

---

## 5. 서버 / API / DB / 프론트 구조 설명

### Front-End

* HTML
* CSS
* JavaScript

역할:

* 사용자 화면 제공
* API 요청 전송
* 데이터 표시

### Back-End

* Node.js

역할:

* 게시글 CRUD 처리
* API 제공
* DB 연동

### Database

#### list_posts

| 컬럼명   | 설명     |
| ----- | ------ |
| id    | 게시글 ID |
| title | 제목     |
| text  | 내용     |

#### video_posts

| 컬럼명 | 설명    |
| --- | ----- |
| id  | 영상 ID |
| url | 영상 링크 |

### API 구조

GET /list_posts

* 게시글 조회

POST /list_posts

* 게시글 생성

PATCH /list_posts/{id}

* 게시글 수정

DELETE /list_posts/{id}

* 게시글 삭제

동일한 방식으로 video_posts API 제공

### 전체 구조

사용자
↓
Front-End
↓ HTTP 요청
Node.js API
↓
Database

## 6. 가장 어려웠던 문제와 해결 과정

### 문제

게시글 수정과 삭제 시 특정 게시글을 정확하게 식별해야 하는 문제가 있었습니다.

### 원인

제목이나 내용만으로는 같은 값이 존재할 수 있어 어떤 게시글을 수정 또는 삭제해야 하는지 구분하기 어려웠습니다.

### 해결 방법

각 게시글에 고유한 id 값을 부여했습니다.

### 결과

* 특정 게시글 조회 가능
* 수정 기능 구현 가능
* 삭제 기능 구현 가능
* 데이터 관리 안정성 향상
